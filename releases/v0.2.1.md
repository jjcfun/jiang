# Jiang v0.2.1

`v0.2.1` 是 `v0.2.0` 之后的 Stage2 bootstrap 更新版。

## Highlights

- Stage2 继续作为默认且唯一的日常自举路径
- LLVM 版本线固定为 `21.1.x`
- 新增初始泛型 `struct` 支持：
  - `@<T> struct Box { ... }`
  - `Box<Int>` 显式实例化
- 新增初始顶层泛型函数支持：
  - `id<Int>(42)` 显式实例化
  - `id(42)` 最小参数推导
- Stage2 回归继续覆盖：
  - `emit-c`
  - `run`
  - `selfhost`
  - `error`
  - `llvm`
  - `llvm-error`

## Bootstrap Meaning

从这个版本开始，推荐把 `v0.2.1` 作为后续主线开发的 bootstrap compiler：

- 使用 release 包中的 `jiang` 自举当前 `compiler/`
- Stage1 不再是默认路径
- Stage2 自举和 selfhost 继续作为主线门槛

## Validation

发布前通过：

- `bash ./script/build_stage2.sh`
- `bash ./script/stage2_run_smoke.sh`
- `bash ./script/stage2_complete_smoke.sh`

## Upgrade Guidance

推荐把 release 提供的 `jiang` 安装到固定位置，例如：

```bash
~/.jiang/toolchains/v0.2.1/bin/jiang
```

然后在构建主线 Stage2 时设置：

```bash
export STAGE2_BOOTSTRAP_STAGE2=$HOME/.jiang/toolchains/v0.2.1/bin/jiang
bash ./script/build_stage2.sh
```

当前泛型能力仍属于第一阶段：

- 已支持泛型 `struct`
- 已支持顶层泛型函数
- 还未覆盖完整的 imported generic instantiation 与更完整的泛型类型族
